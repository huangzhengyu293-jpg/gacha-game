## 📖 详细场景说明

### 场景 1: 经典模式 - 4人 - 纯净对战

**目的**: 测试最基础的4人对战流程

**配置**:
- 游戏模式: `classic` (经典)
- 对战类型: `solo` (个人)
- 玩家数量: 4人
- 特殊规则: 无

**轮次设计**:
```
第1轮 (不给糖就捣蛋宝可梦):
  - p1: 仙子伊布 VMAX (epic, $735) ✅ 最高
  - p2: Zekrom ex (rare, $588)
  - p3: 雷希拉姆 (uncommon, $255)
  - p4: 精灵球 (common, $1) ✅ 最低

第2轮 (高价值混合礼包):
  - p1: 超级仙子伊布 (epic, $1200) ✅ 最高
  - p2: Zekrom VMAX (rare, $650)
  - p3: 喷火龙GX (uncommon, $400)
  - p4: 能量卡 (common, $50) ✅ 最低

第3轮 (入门礼包):
  - p1: 稀有训练师 (rare, $80) ✅ 最高
  - p2: 进化卡 (uncommon, $35)
  - p3: 基础卡 (common, $5)
  - p4: 基础卡 (common, $5)
```

**预期结果**:
- 获胜者: `p1` (总计 $735 + $1200 + $80 = $2015)
- 最终排名: p1 > p2 > p3 > p4

**测试要点**:
- ✅ 无二段动画，所有老虎机一次到位
- ✅ 每轮回正时立即更新金额和round记录
- ✅ 最终显示p1为获胜者

### 场景 2: 经典模式 - 4人 - 传说降临

**目的**: 测试二段动画（传说物品）的正确触发和处理

**配置**:
- 游戏模式: `classic`
- 对战类型: `solo`
- 玩家数量: 4人
- **关键**: 第2轮有2人抽到传说物品

**轮次设计**:
```
第1轮 (不给糖就捣蛋宝可梦):
  - p1: 仙子伊布 VMAX (epic, $735)
  - p2: Zekrom ex (rare, $588)
  - p3: 雷希拉姆 (uncommon, $255)
  - p4: 精灵球 (common, $1)

第2轮 (传说宝可梦礼包) ⭐ 关键轮次:
  - p1: 离子（传说） (legendary, $250) 🎰 需要二段！
    第1段: 金色占位符
    第2段: 离子
  - p2: 莉莉（传说） (legendary, $700) 🎰 需要二段！
    第1段: 金色占位符
    第2段: 莉莉
  - p3: 史诗卡 (epic, $500) ✅ 一段到位
  - p4: 普通卡 (common, $5) ✅ 一段到位

第3轮 (不给糖就捣蛋宝可梦):
  - p1: 喷火龙ex (rare, $385)
  - p2: 火箭队的超梦前任 (epic, $685)
  - p3: 雷希拉姆 (uncommon, $255)
  - p4: 盖诺赛克特 (common, $79)
```

**时间轴分析**:
```
第2轮时间线:
├─ 0-5s:   第1段 spinning (p1/p2 → 金色, p3/p4 → 最终结果)
│          5s: p3/p4 回正 ✅ 记录结果
├─ 5-6s:   第1段 complete (展示金色占位符)
├─ 6-11s:  第2段 spinning (p1/p2 → 传说物品)
│          11s: p1/p2 回正 ✅ 记录最终结果
└─ 11-12s: 第2段 complete (展示传说物品)
```

**预期结果**:
- 获胜者: `p2` (总计 $588 + $700 + $685 = $1973)
- 最终排名: p2 > p1 > p3 > p4

**测试要点**:
- ✅ p1 和 p2 的老虎机转两次
- ✅ 第1段结束(5s)时，只有p3和p4的结果被记录
- ✅ 第2段结束(11s)时，p1和p2的最终结果被记录
- ✅ p3和p4在第2段期间保持静止，显示第1段结果
- ✅ 金色占位符正确渲染

### 场景 3: 淘汰模式 - 4人 - 步步惊心

**目的**: 测试每轮淘汰机制

**配置**:
- 游戏模式: `elimination`
- 对战类型: `solo`
- 玩家数量: 4人 → 3人 → 2人
- **规则**: 每轮结束后淘汰当轮最低分玩家

**轮次设计**:
```
第1轮 (4人参与):
  - p1: $735
  - p2: $588
  - p3: $255
  - p4: $1 ← 🔥 淘汰！

第2轮 (3人参与):
  - p1: $1200
  - p2: $650
  - p3: $50 ← 🔥 淘汰！

第3轮 (2人参与):
  - p1: $700 (二段传说) ← 🏆 冠军
  - p2: $50 ← 🥈 亚军
```

**时间轴分析**:
```
第1轮结束 → 展示淘汰动画 → p4标记为淘汰 → 进入第2轮
第2轮结束 → 展示淘汰动画 → p3标记为淘汰 → 进入第3轮
第3轮结束 → 展示获胜者
```

**预期结果**:
- 获胜者: `p1`
- 淘汰顺序: `p4` (第1轮) → `p3` (第2轮)
- 最终排名: p1 (冠军) > p2 (亚军) > p3 (第三) > p4 (第四)

**测试要点**:
- ✅ 第1轮结束后显示p4淘汰动画
- ✅ 第2轮只渲染3个老虎机
- ✅ 第2轮结束后显示p3淘汰动画
- ✅ 第3轮只渲染2个老虎机
- ✅ p4和p3在被淘汰后显示灰色/透明效果
- ✅ 最终正确显示冠军和排名

## ✅ 检查清单

请逐一检查以下内容：

### 通用规则
- [ ] 所有场景的参与者ID唯一且一致
- [ ] 所有itemId都对应到卡包中的实际物品
- [ ] qualityId与物品的qualityId匹配
- [ ] needsSecondSpin标记正确（只有legendary才为true）

### 时间轴相关
- [ ] 每轮的持续时间计算正确
- [ ] 二段动画的时间线正确（两个spinning事件）
- [ ] 回正时间点触发金额更新
- [ ] 展示阶段不触发更新

### 游戏模式规则
- [ ] **Classic**: 累计最高分获胜
- [ ] **Elimination**: 每轮淘汰最低分，最后留下的获胜
- [ ] **Jackpot**: 最高累计分获得所有奖池
- [ ] **Sprint**: 基于品质积分，最高分获胜
- [ ] **Team**: 团队成员分数累加

### 特殊规则
- [ ] **Fast Mode**: 动画时间缩短
- [ ] **Last Chance**: 最后一轮高价值
- [ ] **Inverted**: 最低分获胜

### UI/UX
- [ ] 二段动画玩家看到金色占位符
- [ ] 非二段玩家在第2段保持静止
- [ ] 淘汰动画正确显示
- [ ] 获胜者烟花效果
- [ ] 团队模式正确分组显示
