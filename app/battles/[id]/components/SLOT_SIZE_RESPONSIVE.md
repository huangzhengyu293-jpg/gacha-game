# 老虎机尺寸智能响应式系统

## 🎯 设计原理

**核心思想**: 将容器宽度**平均分配**给每个老虎机，根据分配到的宽度选择合适的尺寸档位。

---

## 📏 尺寸档位

| 档位 | 尺寸 | 触发条件 |
|------|------|---------|
| 1 | **160px** | 平均分配宽度 ≥ 160px |
| 2 | **140px** | 平均分配宽度 ≥ 140px 且 < 160px |
| 3 | **120px** | 平均分配宽度 ≥ 120px 且 < 140px |
| 4 | **90px** | 平均分配宽度 < 120px |

---

## 🧮 计算逻辑

### 公式
```
平均分配宽度 = 容器宽度 / 老虎机数量

if (平均分配宽度 >= 160) → 使用 160px
else if (平均分配宽度 >= 140) → 使用 140px
else if (平均分配宽度 >= 120) → 使用 120px
else → 使用 90px
```

### 适用范围
- ✅ **仅适用于**: 单人模式（solo）且 2-4 人
- ❌ **其他模式**: 使用默认布局，不调整尺寸

## 💻 代码实现

```typescript
const calculateItemSize = () => {
  const containerWidth = container.offsetWidth;
  
  // 平均分配给每个老虎机的宽度
  const avgWidthPerSlot = containerWidth / playersCount;
  
  // 根据分配的宽度选择最合适的档位
  if (avgWidthPerSlot >= 160) {
    setItemSize(160);
  } else if (avgWidthPerSlot >= 140) {
    setItemSize(140);
  } else if (avgWidthPerSlot >= 120) {
    setItemSize(120);
  } else {
    setItemSize(90);
  }
};

// 使用 ResizeObserver 监听容器宽度变化
const resizeObserver = new ResizeObserver(calculateItemSize);
resizeObserver.observe(container);
```

## 🎯 关键点

1. **容器 = max-w-screen-xl (1280px) 自适应**
   - 大屏: 1280px
   - 中屏: 100% - 32px (padding)
   - 小屏: 100% - 32px

2. **平均分配**
   - 忽略 VS 装饰符和间距
   - 简单除法：容器宽度 / 老虎机数量

3. **档位阈值**
   - 160px: 高清大图
   - 140px: 标准尺寸
   - 120px: 紧凑布局
   - 90px: 最小可用
