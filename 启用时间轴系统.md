## 🚀 如何启用（2个方法）

### 方法1：直接访问（无需修改代码）

**访问时间轴版本**：
```
http://localhost:3000/battles/test-id?useTimeline=true
```

**模拟从X秒进入**：
```
http://localhost:3000/battles/test-id?simulateJoinTime=10
```

> 这个方法不需要修改任何代码，直接通过 URL 参数切换！

---

### 方法2：修改入口文件（永久启用）

打开 `app/battles/[id]/page.tsx`

**在第1行添加**：
```typescript
export { default } from './page-wrapper';
```

**然后注释掉或删除后面所有代码**。

保存后，所有对战页面访问都会：
- 默认使用原版（状态机）
- 添加 `?useTimeline=true` 或 `?simulateJoinTime=X` 自动切换到时间轴版本
