# ⚡ 立即开始测试！

## 🎯 只需3步

### 第1步：确保项目运行中

```bash
npm run dev
```

### 第2步：访问测试页面

复制这个链接到浏览器：

```
http://localhost:3000/battles/timeline-test?packIds=1,2,3&players=4
```

### 第3步：观察效果

页面会：
1. 自动填满参与者（机器人）
2. 倒计时 3、2、1
3. 老虎机自动转动
4. 显示结果

**打开控制台（F12）查看详细日志！**

---

## 🎮 测试按钮

页面底部绿色边框的面板里有这些按钮：

### 必试按钮

1. **⏸️ 暂停** - 点击后，时间会停止，老虎机会停止转动
2. **▶️ 播放** - 点击后，继续播放
3. **⏩ 跳到5秒** - 直接跳到5秒的位置
4. **🎬 测试中途加入** - 模拟10秒前开始，直接跳到对应位置

## 🐛 如果还是没有老虎机

刷新页面，查看控制台是否有：

```
❌ battleData.packs 是空的！无法生成结果！
```

**如果看到这个**，说明礼包API还没返回数据。

**临时解决方案**：页面已经添加了模拟数据兜底，稍等几秒让API加载完成，或者刷新页面。
